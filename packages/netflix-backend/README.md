# Netflix Backend

Backend Node.js/TypeScript type Netflix-like.
Ce projet est le systeme observe par le Production Reality Lab.

Structure:
Couche Documentation
Domain quoi  
Application pourquoi
Infrastructure comment
Observability quand / combien

Ce projet est avant tout une plateforme de services backend.
Les interfaces utilisateurs (HTML, CLI, batchs, autres services)
sont considérées comme des clients externes et ne contiennent
aucune logique métier.
